Ship one stable-ABI (abi3) wheel per arch instead of four. The cffi extension is built with `py_limited_api=True`, so a single `cp310-abi3-...` wheel covers Python 3.10 through 3.13. The previous `cibuildwheel` matrix produced four wheels all tagged `cp310-abi3-...` and silently overwrote each other, leaving whichever the matrix happened to finish last. The release `test-wheels` job now imports the single abi3 wheel under each Python version (3.10 / 3.11 / 3.12 / 3.13) and runs the integration suite to verify the abi3 promise.
