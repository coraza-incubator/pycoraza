Add `extract_client_ip=` to the Flask and Starlette/FastAPI middleware. Accepts a preset name (`"cloudflare"`, `"xff_first"`, `"xff_last"`) or a callable. The new `pycoraza.trusted_proxy()` helper builds a CIDR-aware extractor that walks `X-Forwarded-For` right-to-left and returns the first untrusted hop — correct for ALB / Nginx / gunicorn-behind-proxy. Default behavior is unchanged.
